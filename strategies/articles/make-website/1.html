Part 1

How to make your very own gaming website

Introduction

Before you start, I assume you already knows how to code HTML, but don't worry if you don't, AI's got everything ^_^

Learning Goal

In this part, you'll learn:

  • Start using GitHub
  • Use repo and add file
  • Publish your website using GitHub Pages

Start Using GitHub

First, go to GitHub and sign up for an account that's NOT YOUR SCHOOL ACCOUNT

Find the "New Repository" button on the left menu, click it.

Use repo and add file

  • For the name, what matters is do you want your website to be [yourname].github.io or [yourname].github.io/[repository name]

  • If you want [yourname].github.io, name the repository as [yourname].github.io
  • If you want [yourname].github.io/[repository name], name the repository as anything you want

  • After naming, make sure to select "Public", nothing else don't really matter, unless you want your website to be copyrighted, added a lisence. For the lisence, I actually don't know anything about it, ChatGPT is the right person thing to ask.

  • Now, click here to download an example game file which is Minecraft 1.8.8.
    You can also download other game files as long as they are HTML files.

  • Go back to your repository page, click "Add file" -> "Upload files"

  • Drag and drop the game file you just downloaded, wait for it to finish uploading, then click "Commit changes" at the bottom.

  • Now change your entire file name to index.html
    You don't have to understand the following things
    A repository, or a repo, will see index.html as the default file, if your file name is something else, for example, eaglercraft-1.12.2.html, then you have to type [yourname].github.io/[repo name]/eaglercraft-1.12.2.html or [yourname].github.io/eaglercraft-1.12.2.html
    If your file name does not end with .html, such as "eaglercraft-1.12.2.sb3", the website is not going to function correctly. If your file name does not have any endings such as "eaglercraft.html" once you enter the link [yourname].github.io/[repo name]/eaglercraft, you are going to download the file.
    If you are uploading an image, then the image file shouldn't be ending with .html, it can be .png, .svg, .jpg, etc. .html is the browser tab code, not a form of image.

Publish your website using GitHub Pages

  • Now, go to "Settings" tab on the top right.
  • Scroll down to "Pages" section in the left menu, it should show a line of text "Deploy from a branch" at the right. Under "Source", select "main" branch, then click "Save".
  • After saving, wait for about 2 minutes, then refresh the page, it should show a message "Your site is ready to be published at https://[yourname].github.io/[repository name]" if you named your repository as [yourname].github.io, then the link will be https://[yourname].github.io. Once you press "visit site" or type the link of your HTML file, you will be able to access the link. It will work on everybody's computer.
  • Congratulations! You have made your very own gaming website!
